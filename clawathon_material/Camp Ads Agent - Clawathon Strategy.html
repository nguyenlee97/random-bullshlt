Insight quyết định

Cuộc thi chấm 100% bằng community voting — KHÔNG chấm kỹ thuật hay chi phí.

BTC nói nguyên văn cuộc thi “hoàn toàn 100% dựa trên voting page”. Toàn bộ nhân viên VNG sẽ tự mở agent của bạn trên voting page và test bằng mock data, rồi đánh giá 3 tiêu chí: tính khả thi · tính hữu dụng · tính hiệu quả

Hệ quả số 1 — Trực quan thắng kỹ thuật

Một agent ai cũng hiểu trong 15 giây sẽ thắng một pipeline kỹ thuật sâu mà chỉ dân Adtima mới hiểu. Bạn không thi với kỹ sư giỏi nhất — bạn thi để được nhiều người bấm vote nhất.

Hệ quả số 2 — Cấm API production là lợi thế

Mọi team đều phải mock dữ liệu. Điều này xóa lợi thế của team có sẵn hệ thống thật và mở khóa use case full-pipeline của bạn: rủi ro phụ thuộc API Adtima nay không còn.

Schema input → output (để dán vào prompt)

Ép agent đọc & sinh đúng các trường này để output ổn định khi voter test:

# INPUT — brief khách (agent phải bóc tách đủ) { "brand": str, # tên brand + ngành "objective": str, # mục tiêu chiến dịch "funnel_stage": "awareness" | "consideration" | "conversion", "target": { "age", "gender", "location", "interests"[], "behaviors"[] }, "budget_vnd": int, "flight": str, "kpi"[], "format"[] } # OUTPUT — agent sinh ra (xuyên suốt 6 bước) { "audience_segment": { "matched"[], "size_est", "gaps"[] }, # bước 3 "setup_plan": { "zones"[], "bid", "schedule", "budget_split" }, # bước 4 "kpi_target": { ... }, # theo funnel "report_eval": [ { "camp_id", "verdict", "reason" } ], # bước 5 "ao_alert": str # bước 6 }
Vì sao tách theo funnel
KPI & cơ chế tối ưu khác hẳn nhau: Awareness chấm reach/CPM/VTR, Conversion chấm CR/CPL/ROAS. Cho agent nhận diện funnel_stage ngay từ brief rồi đổi bộ KPI & logic setup tương ứng — đây là điểm “thông minh nghiệp vụ” voter đánh giá cao.

Hợp đồng bắt buộc với AgentBase Runtime

  • Container nghe 0.0.0.0:8080 + có GET /health → 200.
  • Không hardcode key — đăng ký trong Access Control, gitignore .env.
  • MaaS base_url: https://maas-llm-aiplatform-hcm.api.vngcloud.vn/v1; truyền model bằng giá trị path.
  • Cân nhắc SDK pip install greennode-agentbase (tự lo /health + context).
  • Deploy nhanh bằng skill pack: /agentbase-wizard → /agentbase-deploy → /agentbase-monitor.
